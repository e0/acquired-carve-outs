# acquired-carve-outs

This is a web scraper that collects the "Carve Outs" from the aquired.fm podcast.

This project uses [Vite+](https://viteplus.dev).

## Run

To install dependencies:

```bash
vp install
```

To run the web server:

```bash
vp dev
```

To seed the data:

```bash
vp run seed
```

To update the data:

```bash
vp run update
```
